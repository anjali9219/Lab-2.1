Student Performance EDA
This project performs Exploratory Data Analysis (EDA) on the Student Performance dataset.

Tools Used
 Python
 Pandas
 Matplotlib
 Seaborn
 Jupyter Notebook
 Dataset
student-mat.csv contains student information such as:

study time
 grades
 absences
 parental education
 Objective
Analyze factors that affect student academic performance.

File
Lab_2.1.ipynb – Jupyter notebook with complete analysis
